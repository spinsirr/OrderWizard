OrderWizard Installation

To install OrderWizard, simply drag the OrderWizard.app icon to the Applications folder.

OrderWizard requires Tesseract OCR to be installed for image text recognition functionality.
If you don't have Tesseract installed, please install it using Homebrew:

    brew install tesseract

After installation:
1. Open the application from your Applications folder
2. Start managing your orders with OCR support and comprehensive tracking

For more information, visit the documentation or contact support.
